# Jitendra Swami | Portfolio 🚀

A sleek, responsive, and high-performance personal portfolio website built to showcase my journey as a Software Developer.

## ✨ Features
- **🌓 Dark/Light Mode**: User-friendly theme toggle with local storage persistence.
- **📱 Fully Responsive**: Optimized for all screen sizes (mobile to desktop).
- **✍️ Typing Animation**: Interactive hero section highlighting core expertise.
- **📜 Scroll Reveal**: Smooth entrance animations for a premium user experience.
- **✉️ Direct Contact**: Integrated form that opens Gmail directly with pre-composed messages.

## 🛠️ Built With
- **HTML5** & **Semantic Tags**
- **CSS3** (Flexbox, Grid, Custom Properties)
- **Vanilla JavaScript** (Intersection Observer API, LocalStorage)
- **FontAwesome** (Icons)

## 📂 Project Structure
- `index.html` - Core structure and content.
- `styles.css` - Custom styling and design system.
- `script.js` - Logic for animations, theme switching, and form handling.
- `Jitendra_resume.pdf` - Professional CV.
